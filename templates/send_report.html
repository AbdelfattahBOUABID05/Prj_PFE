Configuration de Google Mail

Pour envoyer des emails via cette application, vous devez utiliser un Mot de passe d'application :

  1. Allez sur votre Compte Google.
  2. Activez la Validation en deux étapes dans l'onglet Sécurité.
  3. Cherchez "Mots de passe d'application" en bas de la page.
  4. Générez un nouveau mot de passe pour "Autre" et copiez le code de 16 caractères.
